# 🎓 Campus Support Portal (College Help Desk)

An interactive, web-based campus support portal designed to streamline issue reporting and management for students and staff.

🚀 **[View Live Demo](https://premrajj53-ui.github.io/campus-support-portal/)**

---

## 📝 Description
This application allows students to dynamically submit technical or administrative support tickets, while providing a dedicated administrative dashboard to track, update, and manage resolutions in real-time. 

## ✨ Features
* **Student Dashboard:** An intuitive form interface for users to lodge support requests (e.g., IT support, hostel maintenance).
* **Admin Portal:** A specialized dashboard view for administrators to track incoming tickets, filter issues, and update statuses.
* **Live Deployment:** Hosted publicly via GitHub Pages for easy, instant access.
* **Data Persistence:** Utilizes client-side browser storage to save, read, and retain ticket data dynamically across sessions.
* **Responsive Layout:** Built with clean CSS to ensure full usability across both desktop and mobile devices.

## 🛠️ Tech Stack
* **Frontend UI:** HTML5, CSS3 (Flexbox/Grid layout)
* **Logic & State Management:** Vanilla JavaScript (ES6+ Asynchronous Logic)
* **Data Storage:** LocalStorage
